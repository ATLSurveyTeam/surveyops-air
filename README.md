# SurveyOps Air

SurveyOps Air is a web application for managing ATL Airport Service Quality (ASQ) survey fieldwork.

## Current Version

**v0.35 MVP**

## Current Capabilities

- Agent workspace
- Manager workspace
- Departure surveys
- Arrival surveys
- Commercial surveys
- Manager PIN access
- Local browser storage
- CSV export
- Surveyor management
- Undo last survey

## Development

Install dependencies:

    npm install

Run the development server:

    npm run dev

Open your browser to:

    http://localhost:5173
